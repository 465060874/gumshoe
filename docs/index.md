Gumshoe User Guide
==================

Overview
--------

Gumshoe intercepts java calls to sockets and captures network statistics.
Data can be logged to a file or viewed live.

[Features](features.md)
-----------------------

- socket I/O analysis
- CPU utilization analysis
- unclosed socket detection  
- file I/O analysis
